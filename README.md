# practica
repositorio para aprender html css y javascripts

## Tarea inicial – Página de productos

Vamos a trabajar en equipo, pero cada uno debe construir su propia base.
Esta actividad es para empezar a dominar HTML desde cero, sin atajos.

---

### 🔹 1. Crear tu primer archivo

Crea un archivo llamado:

```
productos.html
```

La idea es que empieces a trabajar fuera del `index.html` y entiendas cómo se construyen páginas independientes.

---

### 🔹 2. Estructura base (escríbela tú)

No copies y pegues sin pensar. Escríbelo y entiende cada parte:

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Productos - Veltrion</title>
</head>
<body>

  <header>
    <h1>Productos</h1>
    <a href="index.html">Volver al inicio</a>
  </header>

  <main>

  </main>

  <footer>
    <p>© 2026 Veltrion</p>
  </footer>

</body>
</html>
```

---

### 🔹 3. Crear productos (lo importante)

Ahora viene lo clave: construir contenido real.

Debes crear **mínimo 5 productos** usando la etiqueta `<article>`.

Ejemplo:

```html
<article>
  <h2>Taladro</h2>
  <p>Precio: $200.000</p>
  <button>Agregar al carrito</button>
</article>
```

---

## Retos

No hagas lo mínimo. Hazlo bien.

Cambia los nombres de los productos (nada de copiar los mismos)
Usa precios diferentes
Crea al menos 5 productos
Comenta tu código explicando qué hace cada parte

---

Objetivo

Entender cómo estructurar contenido en HTML usando:

* `article`
* `img`
* `p`
* `button`
